# Innstala - Soporte Remoto

Esta es la herramienta oficial de asistencia técnica de **Innstala**. 
Diseñada para ofrecer una conexión segura, rápida y sencilla entre nuestros técnicos y nuestros clientes.

## Características
- Conexión cifrada de punto a punto.
- Sin necesidad de instalación (versión portable).
- Optimizado para Windows y Linux (Debian).

*Este software está basado en el proyecto de código abierto [RustDesk](https://github.com/rustdesk/rustdesk) bajo licencia AGPL-3.0.*
